## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AndrewCrescencio/andrewcrescencio-blog.git
   ```

2. Navigate to the folder
   ```sh
   cd andrewcrescencio-blog
    ```
     
3. Install dependencies
   ```sh
   yarn install
   ```




+ ### Development server
    ```
    yarn dev
    ```
    
+ ### Storybook
    ```
    yarn storybook
    ```
